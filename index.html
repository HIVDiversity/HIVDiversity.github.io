WebWidgets: javascript for bioinformatics

Here we post links to useful javascript applications that will run locally in your browser.

  • Longitudinal Divergence widget for generating heatmaps and escape logos.
  • Tree Illustrator widget for illustrating trees.
  • Phylo-Tagger widget for lineage tagging of phylogenetic trees.
  • Alignment Clustering widget for clustering aligned sequences and viewing the clustering on a tree.

These widgets are experimental, please feel free to contact hugh.murrell@gmail.com if you spot any bugs.
